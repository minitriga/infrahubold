The Python SDK (Async or Sync) client can be configured using an instance of the Config class.

<Tabs groupId="async-sync">
  <TabItem value="Async" default>
  ```python
  from infrahub_sdk import Config, InfrahubClient
  config = Config(address="http://infrahub:8080", api_token="123-xyz-invalid-token")
  client = await InfrahubClient.init(config=config)
  ```
  </TabItem>
  <TabItem value="Sync">
  ```python
  from infrahub_sdk import Config, InfrahubClientSync
  config = Config(address="http://infrahub:8080", api_token="123-xyz-invalid-token")
  client = InfrahubClientSync.init(config=config)
  ```
  </TabItem>
</Tabs>

The following settings can be defined in the Config class

## address

**Property**: address  
**Description**: The URL to use when connecting to Infrahub.  
**Type**: `string`  
**Default value**: http://localhost:8000  
**Choices**:   
**Environment variable**: INFRAHUB_SDK_ADDRESS  
## api_token

**Property**: api_token  
**Description**: API token for authentication against Infrahub.  
**Type**: `string`  
**Default value**:   
**Choices**:   
**Environment variable**: INFRAHUB_SDK_API_TOKEN  
## echo_graphql_queries

**Property**: echo_graphql_queries  
**Description**: If set the GraphQL query and variables will be echoed to the screen  
**Type**: `boolean`  
**Default value**: False  
**Choices**:   
**Environment variable**: INFRAHUB_SDK_ECHO_GRAPHQL_QUERIES  
## username

**Property**: username  
**Description**: Username for accessing Infrahub  
**Type**: `string`  
**Default value**:   
**Choices**:   
**Environment variable**: INFRAHUB_SDK_USERNAME  
## password

**Property**: password  
**Description**: Password for accessing Infrahub  
**Type**: `string`  
**Default value**:   
**Choices**:   
**Environment variable**: INFRAHUB_SDK_PASSWORD  
## recorder

**Property**: recorder  
**Description**: Select builtin recorder for later replay.  
**Type**: `enum`  
**Default value**: none  
**Choices**: none,json  
**Environment variable**: INFRAHUB_SDK_RECORDER  
## custom_recorder

**Property**: custom_recorder  
**Description**: Provides a way to record responses from the Infrahub API  
**Type**: `string`  
**Default value**:   
**Choices**:   
**Environment variable**: INFRAHUB_SDK_CUSTOM_RECORDER  
## default_branch

**Property**: default_branch  
**Description**: Default branch to target if not specified for each request.  
**Type**: `string`  
**Default value**: main  
**Choices**:   
**Environment variable**: INFRAHUB_SDK_DEFAULT_BRANCH  
## default_branch_from_git

**Property**: default_branch_from_git  
**Description**: Indicates if the default Infrahub branch to target should come from the active branch in the local Git repository.  
**Type**: `boolean`  
**Default value**: False  
**Choices**:   
**Environment variable**: INFRAHUB_SDK_DEFAULT_BRANCH_FROM_GIT  
## mode

**Property**: mode  
**Description**: Default mode for the client  
**Type**: `enum`  
**Default value**: default  
**Choices**: default,tracking  
**Environment variable**: INFRAHUB_SDK_MODE  
## requester

**Property**: requester  
**Description**:   
**Type**: `string`  
**Default value**:   
**Choices**:   
**Environment variable**: INFRAHUB_SDK_REQUESTER  
## timeout

**Property**: timeout  
**Description**: Default connection timeout in seconds  
**Type**: `integer`  
**Default value**: 10  
**Choices**:   
**Environment variable**: INFRAHUB_SDK_TIMEOUT  
## transport

**Property**: transport  
**Description**: Set an alternate transport using a predefined option  
**Type**: `enum`  
**Default value**: httpx  
**Choices**: httpx,json  
**Environment variable**: INFRAHUB_SDK_TRANSPORT  
## sync_requester

**Property**: sync_requester  
**Description**:   
**Type**: `string`  
**Default value**:   
**Choices**:   
**Environment variable**: INFRAHUB_SDK_SYNC_REQUESTER
